# AGENTS.md

## README 運用ルール
- `README.md` または `README.ja.md` のどちらかを更新する場合、必ずもう片方も同一内容で同時更新すること（英語/日本語の表現差のみ許容）。
- 片方だけの情報追加・削除・修正を禁止する。
- 差分作成時は、README 2ファイルの見出し構成・コマンド例・オプション説明・注意事項が一致していることを更新者が確認してから完了とする。
- 上記ルール違反の変更は受け入れない。

## サブコマンド運用ルール
- 新規に `src/git-*` を追加する場合、ファイル作成時点で実行権限を付与した状態（`chmod +x`）でコミットすること。
- 実行権限の付与をインストーラー処理に依存させないこと。
