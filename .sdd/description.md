# 実装したい機能

## 概要

gitで利用できるサブコマンドを実装します

## 詳細

### ファイル名ルール

`git-***`

というファイル名で実行形式またはスクリプトを作成してパスを通すことで、`git ***` として実行することができる。

### 前提条件

一部機能はfzfを必須とする。
そのためfzfがインストールされていない環境では動作しないようにする。

### 実装する機能

#### git latest [branch]

指定したブランチに切り替え、リモートにある最新の状態に更新する。
このコマンドの後に `git switch -` で切り替え前のブランチに戻せるようにしたいので、ブランチの履歴を途中で追加しないようにする。

#### git local-branch　[branch-pattern]

ローカルブランチのうち `branch-pattern` に一致するブランチ名を選択することで、そのブランチに切り替える。
ブランチの選択はfzfによる絞り込みで行う。
`branch-pattern` を指定しない場合は全ローカルブランチを対象に絞り込む。

#### git merged

ローカルブランチのうち現在のブランチにマージ済のブランチの一覧を出力する。
`-clean` のオプションをつけることで、マージ済のブランチを全て削除する。

#### git last

最後のコミットの変更差分を出力します。
`git last -1`, `git last -2`... と `-n(正の整数)` を渡すことで、n個前のコミットの差分を出力します。
