# 機能概要
セットアップの記述を削除する

## 背景
GitHub Issue #6
https://github.com/meian/git-subcommands/issues/6

## 要求内容（Issue原文）
インストーラーを追加したこととサブコマンド作成時に実行権限をつけるようにルールづけたことでセットアップの記述が不要になったので削除する

## スコープ
- In Scope: README.md / README.ja.md から不要になったセットアップ記述を削除する
- Out of Scope: セットアップ手順そのものの実装変更、インストーラー挙動の変更

## 受け入れ観点（初版）
- [ ] README.md と README.ja.md のセットアップ記述が削除され、内容が整合している
- [ ] インストーラー追加と実行権限ルールに矛盾しない説明になっている
- [ ] 変更がドキュメント更新に限定され、既存機能の実装に影響しない
