# Product Overview

## プロダクト概要
`git-subcommands` は、`git <subcommand>` 形式で利用できる独自サブコマンドを提供するプロジェクトです。
機能定義は `.sdd/description.md` に整理され、`git-*` 形式のサブコマンド実装とテストがリポジトリに配置されています。

## 主要機能
- `git latest [branch]`：指定ブランチに切り替え、リモートの最新状態に更新する
- `git local-branch [branch-pattern]`：ローカルブランチを `fzf` で絞り込み選択して切り替える
- `git merged [-clean]`：現在ブランチにマージ済みのローカルブランチ一覧を表示し、`-clean` 指定時は削除する
- `git last [-n]`：最後のコミット、または `-n` 指定で n 個前コミットの差分を表示する

## 対象ユースケース
日常的に Git ブランチ操作を行う開発者が、ブランチ更新・選択・整理を短いコマンドで効率化したい場面。

## 価値提案
反復的な Git 操作をサブコマンド化し、ブランチ運用の手間を減らします。
特に `git local-branch` は `fzf` によるインタラクティブ選択を前提とし、ブランチ切り替え体験を高速化します。
